# AI Agents Context

## Project
Workout planner web app built with Preact and Vite.

## Tech Stack
- **Framework**: Preact 10.27
- **Build Tool**: Vite 7.2
- **Dev Server**: Configured for LAN access (0.0.0.0:5173)

## Structure
- `src/components/` - Auth and workout logging components
- `src/app.jsx` - Main app component
- `vite.config.js` - Vite configuration with LAN server settings

## Commands
- `npm run dev` - Start dev server
- `npm run build` - Production build
- `npm run preview` - Preview production build
